# Блочный тариф (по умолчанию)

Считается заново каждый календарный месяц. Сум за кВт·ч:

| Блок | кВт·ч в месяц | Цена |
|---|---|---|
| 1 | 0–200 | 650 |
| 2 | 201–500 | 900 |
| 3 | 501–1000 | 1100 |
| 4 | 1001–5000 | 1600 |
| 5 | 5001–10000 | 1900 |
| 6 | 10001+ | 2200 |

Меняются в `index.html` → `DEFAULT_TARIFF`; пользователь может править на сайте
(сохраняется в localStorage его браузера).

Эталонные проверки (используются в тестах):
- 119,27 кВт·ч → 77 526 сум
- 391,06 кВт·ч → 301 954 сум
- 12 000 кВт·ч → 21 250 000 сум
