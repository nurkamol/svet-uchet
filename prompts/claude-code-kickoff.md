# Промпт: старт работы в Claude Code

Откройте терминал в папке репозитория, запустите `claude` и вставьте:

---

Прочитай CLAUDE.md и docs/ — это проект «Свет·Учёт», анализатор выгрузок
электросчётчика в одном index.html. Прежде чем что-то менять:

1. Запусти тесты: `cd tests && npm install && npm test` — все должны пройти. Это точка
   отсчёта: если что-то красное до твоих правок, скажи мне, а не чини молча.
2. Открой samples/demo.csv и убедись, что понимаешь формат данных (docs/data-format.md).
   Там же — ловушка SheetJS с запятой и dd/mm: на ней уже один раз обожглись.

Правила: ядро расчётов живёт между маркерами «ЯДРО РАСЧЁТОВ» в index.html — маркеры
не трогать, тесты их извлекают текстом; данные пользователя не покидают браузер;
после любого изменения ядра — тесты; интерфейс на трёх языках, строки — только в словаре
STR (ключ обязан быть во всех трёх), не хардкодить; цвета — только токенами из :root,
иначе развалится тёмная тема.

Проверяй результат, а не намерение: тесты вида «расход ≥ 0» проходят и на испорченных
данных. Сверяй конкретные значения и, если правил интерфейс, посмотри на него в браузере.

Первая задача — возьми верхний незакрытый пункт из docs/roadmap.md
[или: опиши свою задачу здесь], предложи план из 3–5 шагов, дождись моего ок,
затем реализуй с тестами и обнови roadmap.
